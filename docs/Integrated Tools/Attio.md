### API key retreival
Once you have an account set up with Attio, you can find your API key by navigating through:
- click your username top left corner
- scroll down to workspace settings
- scroll down to developers (under workspace)
- under 'Access tokens', create a new token. 
- assign 'Read-write' permissions to all 
- naming it launchyfi_token is reccommended but not required
- Save changes
- copy the API key
- in the Launchyfi website, navigate to the integrations tab
- add an integration
- select attio from the services list, paste the API key in the box, and save!
