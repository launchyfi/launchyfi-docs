## Setup

### HubSpot setup
if you don't already have an account with HubSpot you can find instructions on how to set up your account at 
[https://app.hubspot.com/signup-hubspot/crm](https://app.hubspot.com/signup-hubspot/crm)

### API key retreival
Once you have an account set up with HubSpot, you can find your Private App Token by navigating through: 

- click on the settings icon on the top tab
- on the sidebar, select 'Legacy apps' in the drop down menu from 'Integrations'
- click on the button that says 'Create Legacy App' on the right
- select 'Private'
- Rename the app and provide a description if you wish
- navigate to the 'scopes' tab, and click 'Add new scope'
- add all CRM scopes and click 'Update'
- 'Create app' in the top left
- Once your app has been created, navigate to 'Auth'
- click on 'show token' and then 'Copy next to 'Acces token'
- in the Launchyfi website, navigate to the integrations tab
- add an integration
- select Hubspot from the services list, paste the Private App Token in the box, and save!
