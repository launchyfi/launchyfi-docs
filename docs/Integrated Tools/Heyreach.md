# HeyReach

![HeyReach](/img/heyreach_logo.png)

## Intro

[HeyReach](https://www.heyreach.io/) pitches itself as a LinkedIn outreach platform to "10x your LinkedIn outbound" with unlimited senders on a fixed cost. It focuses on safe, scalable multi‑account rotation, a unified inbox for managing replies, step‑based sequence automation (connections, messages, profile views), and tight GTM‑stack integrations (e.g., Clay, HubSpot, Zapier, webhooks/API), plus an MCP to plug in AI agents.


## Setup

### API key retreival
Once you have an account set up with HeyReach, you can find your API key by navigating from [https://app.heyreach.io/app/dashboard](https://app.heyreach.io/app/dashboard):
- sclick on the integrations icon on the left sidebar
- Next to 'HeyReach API' click 'Get API key'
- Generate and copy the API key. 
- in the Launchyfi website, navigate to the integrations tab
- add an integration
- select Heyreach from the services list, paste the API key in the box, and save!
